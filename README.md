# 🚀 Google Devs Portal (Crudcito)

¡Bienvenido al **Google Devs Portal**! Una aplicación móvil desarrollada en **Flutter** diseñada para gestionar perfiles de desarrolladores de alto nivel utilizando **Material 3** y conectada a la nube mediante la API REST de **Supabase**. 💻✨

## 🌟 Características Principales

*   **⚡ Arquitectura Limpia:** Separación de UI y lógica de negocio mediante el Patrón Repositorio.
*   **☁️ Conexión a la Nube (API):** Consumo de una API REST autogenerada en Supabase (PostgreSQL) para persistencia real de datos.
*   **🎨 Diseño Material 3:** Interfaz moderna, limpia e interactiva inspirada en los lineamientos de Google.
*   **🔄 CRUD Completo:**
    *   **C**reate: Añade nuevos desarrolladores al equipo.
    *   **R**ead: Visualiza el perfil y los talentos de cada programador.
    *   **U**pdate: Modifica años de experiencia, salarios o roles.
    *   **D**elete: Elimina registros en tiempo real desde la nube.

## 🛠️ Tecnologías Usadas

*   [Flutter](https://flutter.dev/) - Framework UI multiplataforma 💙
*   [Dart](https://dart.dev/) - Lenguaje de programación 🎯
*   [Supabase](https://supabase.com/) - Backend-as-a-Service (BaaS) y API REST 🐘

## 🚀 Cómo ejecutarlo localmente

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Hazielcode/app-de-flutter-con-API.git
   ```
2. Entra a la carpeta del proyecto e instala las dependencias:
   ```bash
   flutter pub get
   ```
3. Ejecuta la aplicación en tu emulador o dispositivo físico:
   ```bash
   flutter run
   ```
